# Интеграция Node.js для yt-dlp

Для корректной работы извлечения субтитров с YouTube необходимо наличие среды выполнения JavaScript. Мы используем портативную версию Node.js.

## Автоматическое скачивание

Вам больше **не нужно** скачивать и устанавливать Node.js вручную. Процесс полностью автоматизирован:

1. При запуске сборки (`npm run build` или `tauri build`) скрипт `build.rs` автоматически скачает `node.exe` для вашей платформы (Windows x64).
2. Файл будет сохранен в папку `src-tauri/bin/` с именем, зависящим от целевой архитектуры (например, `node-x86_64-pc-windows-msvc.exe`).
3. Tauri автоматически включит этот файл в финальный дистрибутив (в качестве `externalBin`).

Приложение будет работать "из коробки" на любом компьютере без необходимости установки Node.js!
