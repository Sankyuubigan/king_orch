ИНСТРУКЦИЯ (СТРОГО ДЛЯ ВЫПОЛНЕНИЯ):
=========================================

### поиск паттерна
Твоя задача — определить Эннеатип и Корневой страх этой проблемы, используя структурный анализ. Внимание! Строго следуй этому алгоритму анализа. Запрещено использовать метод "совпадения ключевых слов". Игнорируй семантический контекст. Важна только механика моей реакции. Не цепляйся за слова и семантику в запросе проблемы. тебе нужно игнорировать объект проблемы и анализировать только **Реактивный Импульс (Вектор)**. тебе нужно найти функциональную механику.

ШАГ 1. ДЕКОНТЕКСТУАЛИЗАЦИЯ
Мысленно убери из данных проблемы клиента все существительные (объекты страха). Оставь только глаголы и связки причинно-следственных связей. Определи, что ДЕЛАЕТ человек внутри своей головы:
- Пытается удержать контроль/границы (Телесный центр)?
- Пытается получить одобрение/идентичность (Сердечный центр)?
- Пытается просчитать будущее/избежать угрозы/понять (Ментальный центр)?

ШАГ 2. ОПРЕДЕЛЕНИЕ ВЕКТОРА
Какой импульс движет реакцией клиента:
- Гнев/Жесткость/Исправление (Типы 8, 9, 1)?
- Стыд/Поиск внимания/Ощущение неполноценности (Типы 2, 3, 4)?
- Страх/Тревога/Ментальное планирование/Уход в голову (Типы 5, 6, 7)?

Чего хочет достичь человек и как выбранный им путь парадоксальным образом усиливает его проблему?

Основываясь СТРОГО на справочнике выше, найди совпадения парадокса петли проблемы клиента с базой знаний. Затем предоставь 3 самых вероятных типов. Каждому из трех типов найденных припиши вероятность (уверенность) совпадения в процентах. Если процент уверенности самого вероятного из вариантов выше 80% то твоим ответом будет этот тип, а если ни один из них не выше 80% то тогда результат будет содержать все три самых вероятных типа (вместе с процентами уверенности у каждого).


Результат должен содержать: эннеатип, согласно которому человек ведёт себя внутри этой проблемы (Номер и Название) + 2 предложения аргументации.

На основе этих трех шагов назови один конкретный тип Эннеограммы, который совпадает не по словам, а по движку мышления. Объясни выбор исключительно через векторы и механику реакций.
