=========================================
ИНСТРУКЦИЯ (СТРОГО ДЛЯ ВЫПОЛНЕНИЯ):
=========================================

### поиск паттерна
Твоя задача — определить паттерн этой проблемы, используя структурный анализ. Внимание! Строго следуй этому алгоритму анализа. Запрещено использовать метод "совпадения ключевых слов". Игнорируй семантический контекст. Важна только механика моей реакции. Не цепляйся за слова и семантику в запросе проблемы. тебе нужно игнорировать объект проблемы и анализировать только **Реактивный Импульс (Вектор)**. тебе нужно найти функциональную механику.

ШАГ 1. ДЕКОНТЕКСТУАЛИЗАЦИЯ
Мысленно убери из данных проблемы клиента все существительные (объекты страха). Оставь только глаголы и связки причинно-следственных связей. Определи, что ДЕЛАЕТ человек внутри своей головы:
- Пытается удержать контроль/границы (Телесный центр)?
- Пытается получить одобрение/идентичность (Сердечный центр)?
- Пытается просчитать будущее/избежать угрозы/понять (Ментальный центр)?

ШАГ 1.5. ФИЛЬТР КОРНЕВОЙ ВАЛЮТЫ (АНТИ-ПОВЕДЕНИЕ)
Перед определением Вектора (Шаг 2) нейросети КАТЕГОРИЧЕСКИ ЗАПРЕЩЕНО делать выводы на основе того, КАК физически защищается человек (например уходит в себя, грустит, агрессирует, суетится). Симптом — это инструмент, а не суть.
Нейросеть обязана задать проверочный вопрос: «Ради спасения какой Базовой Валюты система включила этот конкретный симптом/поведение?»
- Валюта 1: ОЦЕНКА / СТАТУС / ЭГО. Человек включает апатию, избегание или болезнь только для того, чтобы не быть униженным, не оказаться лузером, не столкнуться со своей неполноценностью на фоне ценного объекта. -> ЭТО СЕРДЕЧНЫЙ ЦЕНТР (Типы 2, 3, 4).
- Валюта 2: ИНФОРМАЦИЯ / ЭНЕРГИЯ. Человек включает апатию или избегание, потому что реальность непредсказуема, и он боится физического истощения, поглощения или катастрофы из-за нехватки данных. -> ЭТО МЕНТАЛЬНЫЙ ЦЕНТР (Типы 5, 6, 7).
- Валюта 3: АВТОНОМИЯ / ГРАНИЦЫ. Человек включает апатию или избегание просто для того, чтобы от него отвалили, не заставляли напрягаться и не нарушали его покой, без привязки к самооценке. -> ЭТО ТЕЛЕСНЫЙ ЦЕНТР (Типы 8, 9, 1).

ШАГ 2. ОПРЕДЕЛЕНИЕ ВЕКТОРА
Какой импульс движет реакцией клиента:
- Гнев/Жесткость/Исправление (Типы 8, 9, 1)?
- Стыд/Поиск внимания/Ощущение неполноценности (Типы 2, 3, 4)?
- Страх/Тревога/Ментальное планирование/Уход в голову (Типы 5, 6, 7)?

Чего хочет достичь человек и как выбранный им путь парадоксальным образом усиливает его проблему?

Основываясь СТРОГО на справочнике выше, найди совпадения парадокса петли проблемы клиента с базой знаний. Затем предоставь 3 самых вероятных типов. Каждому из трех типов найденных припиши вероятность (уверенность) совпадения в процентах. Если процент уверенности самого вероятного из вариантов выше 80% то твоим ответом будет этот тип, а если ни один из них не выше 80% то тогда результат будет содержать все три самых вероятных типа (вместе с процентами уверенности у каждого).


Результат должен содержать: паттерн, согласно которому человек ведёт себя внутри этой проблемы (Номер и Название) + 2 предложения аргументации.

На основе этих трех шагов назови один конкретный паттерн, который совпадает не по словам, а по движку мышления. Объясни выбор исключительно через векторы и механику реакций.









ПРАВИЛО КРИТИЧЕСКОЙ НЕХВАТКИ ДАННЫХ:
Если во входных данных от пользователя категорически не хватает деталей для уверенного вывода (например, слишком короткое описание, не ясна эмоция, соматика не подходит ни под один паттерн), ЗАПРЕЩАЕТСЯ выдумывать и галлюцинировать результат.
